# database design and sql project

## Overview
This project demonstrates the design and implementation of a relational database and the use of SQL queries to extract information. 
The project was completed as part of my Higher Diploma in Software Development.

## Project Objectives
- Design a relational database schema
- Create an Entity Relationship Diagram (ERD)
- Implement the database using MySQL
